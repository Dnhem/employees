## 	:toolbox: Getting Started

This project uses NPM as package manager

<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  https://github.com/Dnhem/neotech-employees.git
```

Go to the directory
```bash
  cd employees
  npm install
  npm run dev
```
